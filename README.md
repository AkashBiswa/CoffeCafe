# CoffeeCafe
Coffecafe is a coffee shop's web-based project. All coffee enthusiasts who enjoy coffeecafe's coffee may subscribe to the page and receive updates related to new offers or menu additions. The admin can access the details of the subscribers and send emails to all of them.

### ---------INSTRUCTIONS FOR RUNNING THE PROJECT------------


1. Extract HTML, CSS and JS folder in a single folder.
2. Open the folder using VS Code and run it.
3. To access the backend use Firebase.
4. Open Firebase and create a project.
5. Create authentication and real time database.
6. Replace the adrdress code with the address code in th JS file.
7. Run it again.
